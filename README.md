# Ajudante-De-Costura
Ajudante de Costura
O Ajudante de Costura é uma ferramenta essencial desenvolvida para facilitar a gestão e organização de profissionais da costura e ateliês. O aplicativo foca na produtividade, permitindo o controle de tarefas e medidas de forma rápida e eficiente diretamente no Android.
